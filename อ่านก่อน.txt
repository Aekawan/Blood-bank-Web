---เอาไฟล์ bloodforlift.sql import ใน  phpmyadmin ก่อนนะ ตั้งชื่อ database ว่า  bloodforlift

โฟล์เดอร์ Backend คือ รวมไฟล์สำหรับเขียน backend เขียนพวก select update delete ลงฐานข้อมูลจะเขียนในนี้ทั้งหมด เวลาดึงข้อมู,มาจะรีเทินข้อมูลออกมาเป็น json string

โฟล์เดอร์ Frontend คือ ui ทั้งหมด โดย ui จะติดต่อ กับ backend โดยใช้ ajax ดึงค่าข้อมูลมาเป็น json แล้วทำการแยกข้อมูลต่างๆมาแสดง

ที่ทำงานได้แล้วตอนนี้ก็มี สมัครสมาชิก  login logout ดูรายการผู้ที่ต้องการเลือด  แบบฟอร์มการขอโลหิต  สถานที่บริจาคโลหิต   (เรียนรู้ยังไม่ได้ทำ)

เวลาใช้งานนะ ต้องกรอกขอมูล พวกสมัครสมาชิก แบบฟอร์ม ต้องกรอกให้ครบทุกช่อง เพระายังไม่ได้ทำเช็คข้อมูลที ถ้ากดตกลง ยืนยันไป มันจะผ่านไป เราไม่ได้ดักที เดี๋ยวมันจะ error ถ้าไม่กรอกให้ครบ

function ที่ใช้ดึงข้อมูล จาก backend มาแสดงบน frontend จะอยู่ ใน โฟล์เดอร์ js/script.js


วิธีใช้งาน 
1.รัน xampp
2.สร้าง database โดยใช้ phpmysqlmin สร้างดาต้าเบสชื่อ  bloodforlift
3. หาแท็บ import แล้วเอาไฟล์  bloodforlift.sql มา import
4.เอาโฟล์เดอร์ booldforlife ไปวางไว้ในใน  ไดร์ฟ c  แล้วไปใน xampp เข้าไปใน htdoc แล้ววางเลย
5.รันหน้าเว็บโดยเข้า localhost/bloodforlife มันจะเข้าไปหน้าแรกให้เอง
6.ลองเล่นก่อน บางฟังก์ชั่นก็ต้อง login ก่อนถึงจะเล่นได้ เช่น ฟังก์ชั่น แบบฟอร์มร้องขอโลหิต
7.สมัครสมาชิก  ต้องหรอกให้ครบทุกช่องเพราะเรายังไม่ได้ดักจับ error ถ้ากรอกไม่ครบกลัวมันจะ bug
8. login เข้าระบบ
9. ลองเล่นตามใจเลย
